# Glowe 💄 *(nome provisório)*
> Agenda e gestão para profissionais autônomos da beleza.

## Sobre o projeto
Este projeto serve para ajudar profissionais autônomos da beleza a organizarem
sua agenda de atendimento, serviços e valores, cuidar do financeiro do seu negócio
em um único ambiente. 

## 🎯 Contexto
Projeto desenvolvido para aprendizado e portfólio, documentando todo o
processo de construção ("building in public").

## ✨ Funcionalidades planejadas (MVP)
- Agenda (agendamentos e eventos)
- Gestão de clientes
- Catálogo de serviços e adicionais
- Orçamento para eventos (com sinal e saldo)
- Dashboard financeiro
